# 🌦️ weather-forecast

A simple and responsive Weather Forecast application that provides real-time weather updates for any city using a weather API.

🚀 Features
🌍 Search weather by city name

🌡️ Displays temperature, humidity, and wind speed

☁️ Shows weather conditions (Clear, Clouds, Rain, etc.)

📅 Real-time data fetching

📱 Responsive UI (works on mobile & desktop)

🛠️ Tech Stack
Frontend: HTML, CSS, JavaScript / React (if you used React)

API: OpenWeather API (or whichever you used)

HTTP Client: Axios / Fetch API
